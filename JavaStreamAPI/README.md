## Overview

This program creates a list of employees and performs multiple stream-based operations.

---

## Features Implemented

1. Find highest salary employee
2. Count employees based on gender
3. Calculate total salary expense department-wise
4. Display top 5 senior employees (by age)
5. Print names of all managers
6. Increase salary by 20% (except managers)
7. Display total employee count

---

## How to Run

Compile and execute:

```bash
javac EmpMain.java
java EmpMain
```
